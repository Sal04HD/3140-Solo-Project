# __MEMORY MATCHING GAME__!
> A Final Solo Project by Sajed Saleh!

# PROJECT EXPLANATION
> *My solo project is a browser-based **Memory Matching Game** built using the follow applications:*
> - HTML
> - CSS
> - JS
> 
> *The goal of the game is for players to flip over the cards and find matching pairs while keeping track of their moves and completion time! The project focuses on core front-end development concepts that allow for proper game interaction. Cards are randomized at the start of each game to create a different experience every time the player restarts.*


> Here are the current features built into this project:
> - Card Flipping and Matching Logic
> - Randomized Card Placement
> - Move Counter
> - Time System
> - Restart Function
> - Win detection and completion message
> - Difficulty Mode (Easy, Medium, Hard)

# HOW TO START
> 1. Download the project files.
> 2. Open the project folder.
> 3. Open index.html in your browser.
